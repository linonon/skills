# 題庫 Markdown 格式

題庫檔案存放在 `data/decks/` 目錄，每個 `.md` 檔案是一個獨立題庫。

## 格式

```markdown
# 題庫名稱

tags: javascript, frontend, basics

---

Q: 問題內容
A: 答案內容

---

Q: 另一個問題
支援多行問題
A: 答案也可以
有多行

---

Q: {{cloze}} 類型題目，用雙大括號標記要填空的部分
A: cloze

---

Q: 多選題
- A) 選項一
- B) 選項二
- C) 選項三
- D) 選項四
A: B
```

## 規則

- 用 `---` 分隔每張卡片
- 每張卡片必須有 `Q:` 和 `A:` 開頭的行
- 檔案開頭可用 `tags:` 定義全域標籤（逗號分隔）
- 卡片類型自動偵測：
  - 含 `{{...}}` → cloze（填空題）
  - 含 `- A)` `- B)` 等選項 → multi_choice（多選題）
  - 其他 → basic（基本問答）
- 支援 Markdown 格式（粗體、程式碼區塊等）
